Welcome to Headwaze on Azure

To get the time for waiting make requests like this:

url:/host/chitown/distance_to_bus_ahead/average_velocity/

eg.

http:/headwaysapi.azurewebsites.net/chitown/10/20

For the compmlete generic use this uri:

url: /host/generic/distance_to_bus_ahead/average_velocity/dee/gee/aych/

eg.

http:/headwaysapi.azurewebsites.net/generic/10/10/5.8/0.8/7/
